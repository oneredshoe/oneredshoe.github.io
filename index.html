Fetch!

Javascript Code Sample; Class Partner Project

Fetch!

Fetch!

Fetch! is the product of a partner project for IGME 330, the web deveoplment class for third years. I took the class a semester early, in my second year, in order to avoid the class clashing with other third year classes. Alongside Declan Behan, a current fourth year, I wworked on this api-based project
Our main motivations were our shared interest in cute animals and supporting rescuing dogs and cats instead of breeder farms. Thus we created Fetch, which searches for random dogs/cats that fit under your search preferences, and shows you a little about the animal as well as where the animal is located. The requirements for the project were to combine two apis, so we combined the Petfinder API with the Google Maps API to deliver cute animals
I worked primarily with the Google Maps API and trying to get that to appear according to returned Petfinder information, as well as tackling the parts of Vue that were troubling Declan
I'll probably add screenshots of the code to the images carousel later, but for now the github is public and linked in the tiny link button

Outlawed

DOM/Javascript Code Sample; Class Project

Outlawed

Outlawed

A cowboy-themed webGL game! My inspiration was the "I wanna be a cowboy baby" vine. The original source code was from a small class assignment, which I then edited to change up the combat methods and sprites to make into something else. My main focus in the code was to change the movement of the enemies to not be bouncing around (which was harder than it sounds), and to make the bullets not follow the direction of the player's forward vector, which was much harder than I thought it would be. I was eventually able to figure both parts out, and my only regret is that I used sprites form the internet instead of creating my own.
